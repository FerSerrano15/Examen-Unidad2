Almacenamiento tipo árbol binario

Los árboles binarios son una estructura de datos jerárquica en la que cada nodo puede tener como máximo dos hijos: un hijo izquierdo y un hijo derecho. Esta estructura es extremadamente versátil y se utiliza en muchos campos de la informática, como la manipulación de datos, algoritmos de búsqueda y optimización.

La terminología básica es:

> Raíz: El nodo superior del árbol

> Nodo: Cada elemento del árbol

> Hijo: Un nodo descendiente de otro nodo

> Hoja: Un nodo que no tiene hijos

> Padre: El nodo que tiene un nodo hijo

Se tiene una clase que recibe el valor y al ser el primero lo coloca al principio del arbol. Cuando se ingresa otro valos este se acomoda a la izquierda o derecha dependiendo de si es mayor o menor al primer número

El metodo insertar: Se encarga de crear un nuevo nodo, si la raiz está vacia lo coloca ahi sino llama al metodo "insertar nodo para ponerlo en la posición correcta"

Con el metodo "nuevoNodo" se compara el valor actual, si es menor va a la izquierda y si es mayor a la derecha hasta encontrar la posicion correcta

Con el metodo "eliminar" se elimina el valor proporcionado por el usuario (para hacerlo y actualizar el arbol utiliza el metodo "eliminarNodo"

El metodo "eliminarNodo" utiliza la siguiente logica:

>> Si el nodo no tiene hijos se elimina directamente

>> Si el nodo tiene un solo hijo se sustituye por ese

>> Si el nodo tiene dos su sucesor (en orden) es decir el nodo más pequeño del lado derecho

El metodo "encontrarSucesor": Se encarga de encontrar un sucesor (en orden) de un nodo. El más pequeño del lado derecho. Recorre el lado derecho hacia abajo para encontrar el valor más bajo a la izquierda

Con el metodo "buscarNodo": Si el valor está vacio o si es igual devuelve el valor. Si es valor es menor busca del lado izquierdo sino se va al derecho

El metodo "preorden": Sirve para recorrer el arbol y organizar el nodo por niveles. Recorre el arbol en orden hacia abajo (incluyendo la raiz, lado izquierdo y derecho) después agrupa los valores por niveles con el arreglo "resultado"

El metodo "mostrar": Usando el metodo "preorden" recorre el arbol y organiza los nodos por niveles para tener de resultado una matriz donde cada subarreglo representa un nivel. Usando un "container" de la pagina HTML va mostrando los valores ingresados, además usando las clases "div" hace posible esta division (además de agregar un color especifico para cada nivel)
